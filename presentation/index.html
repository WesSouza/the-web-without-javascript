A Bit of History

  • "a large hypertext database with typed links"
    –Tim Berners-Lee, 1989
  • The web combined HTTP and HTML to enable documents and other content to link to each other
  • WorldWideWeb, the first web browser, was released in 1990
  • JavaScript was introduced later, in 1995

JavaScript

  • Netscape saw the need of a glue language to enhance the web
  • Sun wanted to position Java as the best language for everything
  • Brendan Eich drafted JavaScript in just 10 days

Fast Forward to Today

  • Complex web applications
  • Web applications passing as desktop software
  • Great development experience through frameworks
  • Web-based games, like Adventures With Anxiety!
